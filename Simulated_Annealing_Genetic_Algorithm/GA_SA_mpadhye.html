(1) Portfolio Optimization Problem

Mean-risk models were developed in the 1950s for portfolio selection problems. Value-at-Risk (VaR) and Conditional Value-at-Risk (CVaR) are the most popular measures of downside risk. Mullen et al. (2011) and Ardia et al. (2011) used DEoptim to find the portfolio weights for which the portfolio has the lowest CVaR and each investment can contribute at most 22.5% to total portfolio CVaR risk.

Y <- c(4.28,4.08,4.42,4.17,4.48,4.3,4.82,4.7,5.11,5.13,5.64,5.56)
 fun = function(x,y) { (sin(10*x)*cos(10*y)+2)/sqrt(x^4+y^4+1) } 
 obj = function(z) {
   fn.call <<- fn.call + 1
  SSXY=sum((X-mean(X))*(Y-mean(Y)))
  SSX=sum((X-mean(X))^2)
  b1= SSXY/SSX
  b0=mean(Y)-b1*mean(X)
  Y_Estimated=X*b1+b0
  Residuals= Y-Y_Estimated
  SSE=sum((Residuals -mean(Residuals))^2)
  out <- SSE
  return(out)
 }
rf <- function(a,b) {
  sum(Y - (a + b*X))^2
}
rfobj <- function(k) {
  rf(k[1],k[2])
}
